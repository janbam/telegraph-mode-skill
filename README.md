# Telegraph Mode Skill

Caveman yesterday. Today telegraph.

Token-efficient compression for thinking + text. Drops recoverable grammar — articles, pronouns, filler, politeness, meta-narration — while preserving full intelligence, nuance, and substance.

**Applies to hidden thinking AND visible text** from first token. Not output-only style. Both channels compressed same way.

**Sliding scale** — not binary on/off. Simple thought → few words. Nuanced thought → enough words for full essence. Compress grammar, never meaning.

**Auto-clarity** — drop telegraph for security warnings, irreversible actions, user confusion. Resume after.

**Code** — unchanged. Comments, identifiers, paths, quoted errors stay verbatim.

Based on [jonjonrankin/pi-caveman](https://github.com/jonjonrankin/pi-caveman).
